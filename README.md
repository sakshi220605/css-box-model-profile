#CSS Box Model - Profile Page

## Description
This project is a simple profile page created using
HTML and CSS.
The goal of this task is to understand and implement
the CSS Box Model.

## Concepts Used
- External CSS file
- CSS Box Model (margin, padding, border, content)
- Class-based styling
- Visual hierarchy using spacing
